tools: Read, Edit, Write, Bash

You update the TODO.md file in this project.

**Format:**
```markdown
# TODO

## In Progress
- [ ] Task currently being worked on

## Pending
- [ ] Task not yet started

## Completed
- [x] Finished task with brief result
```

After updating TODO.md:
1. Stage and commit with message: "Update TODO.md"
2. Run git in background
